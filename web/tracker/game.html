Kurzanleitung

1. Spieler auswählen

Spieler im Kader antippen — wird gelb markiert. Nochmal tippen zum Abwählen.

2. Aktion erfassen

Eines der Aktionsfelder unten antippen. Goalie: 🧤 Parade, ⭐ Mega Parade, 🔑 Schlüsselpass, ⚠️ Fehlauswurf. Feldspieler: 🔄 Ballgewinn, 🛡️ Abwehr, 🔑 Schlüsselpass, 🎯 Torschuss, ⚠️ Fehlpass.

3. Goal erfassen

Torschützen wählen → grünen ⚽ Goal-Button tippen → Assistspieler antippen. Falls kein Assist: «Kein Assist» tippen.

4. Gegengoal erfassen

Roten 🔴 Gegengoal-Button tippen → Grund auswählen oder «Kein Grund» tippen. Kein Spieler muss ausgewählt sein.

5. Scout auswählen

Oben im Dropdown den eigenen Namen wählen, damit später klar ist wer was erfasst hat.

6. Kein Netz? Kein Stress.

Einträge werden lokal gespeichert und automatisch gesendet, sobald wieder eine Verbindung besteht. Der rote Badge zeigt an wie viele noch ausstehend sind.

7. Falsch getippt?

Die letzte Aktion mit ↩ Rückgängig korrigieren — sendet eine Korrekturmeldung ans Sheet.
